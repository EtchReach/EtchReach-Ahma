# Enable Ahma

## Folder structure

```tree
├── bike_detector # Flutter mobile application
├── CameraWebServer # Arduino code to be run on the ESP32-Cam
├── CameraWebServerExpress # Express javascript server that mocks the ESP32-Cam API, to be run on computer
├── TOF # Legacy TOF sensor code
└── raspberry_pi # Legacy raspberry pi code for legacy device
```
